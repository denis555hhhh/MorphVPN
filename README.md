# MorphVPN Website

Современный сайт для продажи VPN сервиса MorphVPN с темным дизайном в стиле Dark Matter.

## Особенности

- 🌙 Темный дизайн с неоновыми цветами
- ⭐ Анимированный фон со звездочками
- 📱 Полностью адаптивный дизайн
- 🚀 Быстрая загрузка
- 💬 Кнопка поддержки Telegram
- 🔒 Страницы политики конфиденциальности и условий использования

## Структура проекта

```
├── index.html          # Главная страница
├── privacy.html        # Политика конфиденциальности
├── terms.html          # Условия использования
├── styles.css          # Основные стили
├── script.js           # JavaScript функции
├── server.js           # Express сервер
├── package.json        # Зависимости Node.js
├── Procfile            # Конфигурация для Railway
└── railway.json        # Конфигурация Railway
```

## Локальный запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите сервер:
```bash
npm start
```

3. Откройте браузер и перейдите на `http://localhost:3000`

## Развертывание на Railway

1. Создайте аккаунт на [Railway.app](https://railway.app)

2. Подключите ваш GitHub репозиторий:
   - Нажмите "New Project"
   - Выберите "Deploy from GitHub"
   - Выберите ваш репозиторий

3. Railway автоматически обнаружит `package.json` и развернет приложение

4. Ваш сайт будет доступен по URL, который предоставит Railway

## Технологии

- HTML5
- CSS3 (с анимациями и градиентами)
- JavaScript (ванильный)
- Node.js + Express
