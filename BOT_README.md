# MorphVPN Telegram Bot

Telegram бот для продажи VPN подписок MorphVPN.

## Возможности

✅ Показ доступных тарифов
✅ Информация о сервисе
✅ FAQ (часто задаваемые вопросы)
✅ Ссылка на сайт для оплаты
✅ Поддержка через Telegram
✅ Красивый интерфейс с кнопками

## Установка

### 1. Установи Python 3.8+

### 2. Клонируй репозиторий
```bash
git clone https://github.com/denis555hhhh/MorphVPN.git
cd MorphVPN
```

### 3. Установи зависимости
```bash
pip install -r requirements.txt
```

### 4. Запусти бота
```bash
python bot.py
```

## Использование

Бот доступен в Telegram по юзернейму: @MorphVPN_bot

### Команды:
- `/start` - Начать работу
- `/help` - Справка
- `/plans` - Показать тарифы
- `/about` - О сервисе
- `/faq` - Часто задаваемые вопросы

### Кнопки:
- 💰 Выбрать тариф
- ℹ️ О сервисе
- ❓ FAQ
- 📞 Поддержка

## Развертывание на сервере

### На Railway:

1. Создай новый проект на Railway
2. Подключи GitHub репозиторий
3. Установи переменную окружения:
   - `TOKEN` = твой токен бота

4. Установи команду запуска:
   ```
   python bot.py
   ```

### На Heroku:

1. Создай `Procfile`:
   ```
   worker: python bot.py
   ```

2. Создай `runtime.txt`:
   ```
   python-3.11.0
   ```

3. Загрузи на Heroku:
   ```bash
   git push heroku main
   ```
